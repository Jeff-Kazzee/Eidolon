## How to Use This Directory

1. **Find your issue category** in the table above
2. **Check the specific guide** for known issues and solutions
3. **Search within guides** for error messages you're seeing
4. **Add new issues** when you discover and solve them

## Contributing to Troubleshooting Docs

When you encounter and solve a new issue:

1. Add it to the appropriate guide
2. Include:
   - **Symptom**: What you observed
   - **Cause**: Why it happened
   - **Solution**: How to fix it
   - **Prevention**: How to avoid it

## Template for New Issues

```markdown
### Issue: [Brief Description]

**Symptom:**
What error message or behavior did you see?

**Cause:**
Why does this happen?

**Solution:**
Step-by-step fix.

**Prevention:**
How to avoid this in the future.

**Related:**
Links to related issues or documentation.
```
